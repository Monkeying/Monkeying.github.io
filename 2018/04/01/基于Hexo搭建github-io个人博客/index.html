###配置github.io
这里面默认电脑已经安装git并拥有github账号(没有的自己安装注册一个10分钟的事，不过要注意起名字）。

github.io本质上是一个基于GitHub的仓库。为了实现将本地的文档推到线上，我们需要现在GitHub账户中创建一个特殊的仓库。
点击个人GitHub主页右上角头像-->your profile-->Repositories-->New
![](./media/img/20180401165029.png '')

仓库的配置如下：（注意！仓库的名称必须是yourusername.github.io)
![](./media/img/20180401165711.png '')
点击Create repository之后仓库创建成功

进入刚创建的github.io仓库，点击settings-->GitHub Pages--> choose a theme可以对主页主题进行修改。此时可以看到如图提示可以通过链接访问个人博客了。
！[](./media/img/20180401170144.png '')
